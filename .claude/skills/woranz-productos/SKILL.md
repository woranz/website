# Woranz Productos

Este skill no define un workflow separado. Su función es cargar la referencia canónica de productos antes de escribir copy, proponer visuales o tomar decisiones de contenido.

## Uso

1. Leé `./PRODUCTOS.md`.
2. Ubicá el producto y segmento correcto.
3. Usá esa ficha como base para cualquier decisión de tono, claims, imagery o priorización de beneficios.

## Regla

Si el pedido toca producto y no leíste `./PRODUCTOS.md`, estás trabajando sin contexto suficiente.
